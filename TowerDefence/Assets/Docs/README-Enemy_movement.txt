READ ME

To make an enemy work, the AI.Navigation package need to be installed.
To do so, go to window > package manager and install AI.Navigation.

If NavMesh crashed with scene instalation and need to be set up again:
Go to window > AI > Navigation(Obsolete).
While having window that popped opened, select wall and platform and check on Navigation Static in "Object" field in Navigation(Obsolete).
Then click Bake in "Bake" field in Navigation(Obsolete).
